# Real College Pilot

- Start with one college pilot, not batch ingestion.
- RequirementCapture is manual and reviewed; no AI parsing in this stage.
- If a rule is not representable, extend the schema/renderer/validator rather than hardcoding a school branch.
- Run requirements report, baseline init, gate, diagnose, authoring-report, privacy scan, package, and package-validate before review.
- Human review is required before moving a sanitized template into the template library.
