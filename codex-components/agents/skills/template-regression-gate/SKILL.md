# Template Regression And Gate

Template gates are conservative onboarding checks.

- Run render, OpenXML validation, format conformance, inspect, layout signature, snapshot, coverage, required parts, and custom property checks.
- Keep reports deterministic and machine-readable.
- Do not claim real-college compliance without human review.
- Do not add forbidden assets or executable files to template packages.
