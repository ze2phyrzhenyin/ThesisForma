# Layout Signature

Layout signatures are approximate OpenXML summaries.

- Extract section setup, styles, tables, figures, equations, fields, notes, bibliography, and custom properties.
- Do not perform screenshot or pixel comparison.
- Keep comparison deterministic.
- Treat threshold failures as regression signals, not proof of semantic content errors.
