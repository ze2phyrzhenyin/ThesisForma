# Template Authoring Quality

The authoring report is the pre-publication quality summary for template authors.

Rules:

- Combine requirement mapping, gate, regression, baseline, coverage, diagnostics, checklist, and next actions.
- Mark readiness conservatively: blocking issues mean notReady.
- Real college templates require human approval even when reports pass.
- Do not add AI parsing, web UI, Word automation, or screenshot diff to this workflow.
- Keep examples fictional.
- Update docs and CLI tests for any report field changes.
