# Negative Fixtures

Negative fixtures are expected failures.

Rules:

- A negative fixture suite passes only when expected codes, severity, and fix hints appear.
- Unexpected success is failure.
- Reports must include expected versus actual values.
- Keep fixtures fictional and focused.
- Negative fixtures verify diagnostics; they are not product examples.
